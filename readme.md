# PHP Calculator 

#### :rocket:  Software Requirement
```bash
PHP version 7.4
Composer
MySQL / MariaDB
```

#### :rocket: Installation

```bash
composer install
Run SQL script calculator.sql
Set database credential on database.php
``` 

#### :rocket: Run Calculator
```bash
Add <number> <number>
Subtract <number> <number>
Multiply <number> <number>
Divide <number> <number>
Pow <base> <exp>
```
